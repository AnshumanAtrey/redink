# assets/

Drop your firm's logo here as `assets/logo.png` (PNG, transparent background recommended, 500–1000 px wide).

The assembler embeds it on the cover page of the final report. If `logo.png` is missing, the cover skips the logo and uses text only.

Other firm-specific images (watermarks, classification banners) can go here too. Reference them from `brand-guidelines.md` Section 1.
